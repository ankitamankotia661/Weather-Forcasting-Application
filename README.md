# Weather Forecast Application

A modern, responsive weather application built with vanilla JavaScript, HTML5, and CSS3 that provides real-time weather data, 7-day forecasts, and interactive visualizations.

## ✨ Features

### Weather Data
- **Real-time Weather**: Current temperature, conditions, wind speed, humidity, and precipitation
- **7-Day Forecast**: Daily weather predictions with min/max temperatures
- **Weather Icons**: Dynamic icons with animations for different weather conditions
- **Air Quality Index**: Display air quality information

### Interactive Features
- **City Search**: Search for any city worldwide using Open-Meteo's geocoding API
- **Geolocation Support**: One-click to get weather for your current location
- **Theme Toggle**: Switch between light and dark modes
- **Quick City Access**: Click on pre-loaded cities for instant weather checks

### Data Visualization
- **Temperature Trend Chart**: Line chart showing 7-day temperature progression
- **Precipitation Forecast Chart**: Bar chart showing precipitation probabilities

### User Experience
- **Responsive Design**: Optimized for all screen sizes (mobile, tablet, desktop)
- **Smooth Animations**: Subtle animations for enhanced user experience
- **Loading States**: Visual feedback during data fetching
- **Error Handling**: User-friendly error messages

## 🛠️ Technologies Used

| Technology | Purpose |

| **CSS3** | Modern styling with CSS variables, flexbox, and grid |
